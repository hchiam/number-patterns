## +10: (and last digit = #)

<img align="center" width="120" src="svg/+10.svg"/>

- The # = the last digit in the number. Landing on a darker square = +10.
- For example, 7 x 5 = 35: last digit is 5, and landing on 3 dark squares = +30.
